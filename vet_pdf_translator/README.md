# vet_pdf_translator

Локальный MVP-проект для перевода **текстового ветеринарного PDF** с английского на русский через **Ollama** (по умолчанию модель `qwen3:8b`).

## Что делает проект

Скрипт:
1. Проверяет доступность Ollama (`http://localhost:11434`) и доступность модели.
2. Извлекает текстовые блоки из PDF через PyMuPDF (без OCR).
3. Сортирует блоки сверху вниз и слева направо.
4. Делит длинные блоки на части.
5. Сохраняет блоки в `work/blocks.jsonl`.
6. Переводит блоки по одному и после каждого шага сохраняет прогресс в `work/translated.jsonl`.
7. При повторном запуске продолжает с места остановки.
8. Делает QA-проверку потенциально опасных мест.
9. Собирает `output/translated.docx` в формате «оригинал слева / перевод справа».

## Требования

- Python 3.10+
- Установленная Ollama
- Локально скачанная модель, например `qwen3:8b`

## Как установить Ollama

Следуйте официальной инструкции для вашей ОС: https://ollama.com/download

После установки проверьте, что сервис доступен:

```bash
ollama --version
```

## Как скачать модель

```bash
ollama pull qwen3:8b
```

## Как установить зависимости

```bash
pip install -r requirements.txt
```

## Как положить PDF

Положите файл сюда:

```text
input/source.pdf
```

## Как запустить

```bash
python translate_pdf_mvp.py --pdf input/source.pdf
```

## Какие файлы появятся

- `work/blocks.jsonl` — извлечённые блоки текста
- `work/translated.jsonl` — результат перевода и статусы по блокам
- `work/qa_report.json` — QA-отчёт по проблемным местам
- `output/translated.docx` — финальный документ

## Как продолжить после сбоя

Просто запустите ту же команду ещё раз:

```bash
python translate_pdf_mvp.py --pdf input/source.pdf
```

Скрипт подхватит существующий `work/translated.jsonl` и не будет повторно переводить уже готовые блоки.

## Как пересоздать извлечение

```bash
python translate_pdf_mvp.py --pdf input/source.pdf --force-extract
```

## Как пересоздать перевод

```bash
python translate_pdf_mvp.py --pdf input/source.pdf --force-translate
```

## Как поменять модель

```bash
python translate_pdf_mvp.py --pdf input/source.pdf --model qwen3:14b
```

## Важное предупреждение

Для ветеринарных текстов обязательна финальная человеческая проверка, особенно:
- дозировок
- диагнозов
- препаратов
- единиц измерения

Автоматический перевод и QA — это помощник, а не замена профильной экспертной валидации.
